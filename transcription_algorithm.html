Coding the Transcription Algorithm

If you recall, in lesson 3 we learned about how DNA turns into RNA during a proccess called transcription. In DNA the nucleotides used are adenine (A), cytosine (C), guanine (G), and thymine (T). RNA uses all these same nucleotides except for thymine and instead uses uracil (U). To convert DNA to RNA we convert A->U, C->G, G->C, and T->A.

To code this algorithm, you must first understand what a for loop is. A for loop is handy because it allows you to run the same code over and over again with a new value. For (hehe!) example, you could code something like this next statement: for each letter in the word "cars", print out the letter. What do you think that means? You're right! It means the letters c, a, r, s would be printed out.

In addition to a for loop we will use what is called an if-else statement. An if statement is what we call a conditional statement. Think of this statement: "if my mom is making cookies, then I want to eat one." The condition is your mom is making cookies and if that is the condition, then you have the outcome of wanting to eat a cookie. Many if statments are followed by an else statement. An else statement is used in the case that the condition of the if statement isn't met. For example, if your mom isn't making cookies then you might say "else I'll be sad."
