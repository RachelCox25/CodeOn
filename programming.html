What is Programming?

Programming is the process of writing computer programs. Think of programming as writing a set of instructions for performing some actions. There are multiple different languages that you can use for programming, just like you could write out instructions in different languages like English, French, or Spanish. Some programming languages are: C++ (C Plus Plus), C# (C Sharp), Java, HTML, CSS, Python, and many more! You are going to be learning Python because it is a great language to learn first.

In programming, we have what are called "data types". A data type could be a number or what we call a string, which is short for a "string of text". To declare a data type you have to set it equal to a variable name, just like in math you may see things like x = 6. See the code editor below to see some examples of data types being declared.

You can manipulate these data types with different commands. Some examples of commands are if-statments, for loops, and print. Print is a command that prints to the console whatever you tell it to. In addition to these commands, you can perform different math functions to your variables like adding (+), subtracting (-), multiplying (x), and dividing (/). We will go more into detail about these commands in the following lessons, but for now use the space below to declare some of your own variables!
